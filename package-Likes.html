Classes summary

Likes_Controller This class contains one likable use, which allows members to like a post

Functions summary

clearLikes Clear the likes log of older actions ... used to prevent a like love fest
increaseTopicLikes Increase the number of likes for this topic.
lastLikeOn

Checks if the member has exceeded the number of like actions they are allowed in a given time period.
likePost

Updates the like value for a post/member combo if there are no problems with the request, such as being a narcissist
likesCount

Return how many likes a user has given or the count of thier posts that have received a like (not the total likes received)
likesPostsGiven Return an array of details based on posts a user has liked
likesPostsReceived

Returns an array of details based on posts that others have liked of this user Creates links to show the users who liked a post
loadLikes

Loads all of the likes for a group of messages Returns an array of message_id to members who liked that post If prepare is true, will also prep the array for template use
messageLikeCount Function to get the number of likes for a message
postLikers Function to load all of the likers of a message
prepareLikes

Prepares the like array for use in the template Replaces the current member id with 'You' if they like a post and makes it first Truncates the like list at a given number and adds in +x others
updateLike Perform a like action, either + or -
