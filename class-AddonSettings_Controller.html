Class AddonSettings_Controller

AddonSettings controller handles administration settings added in the common area for all addons in admin panel.

What it does: - Some addons will define their own areas, but for simple cases, when you have only a setting or two, this area will allow you to hook into it seamlessly, and your additions will be sent to admin search and otherwise benefit from admin areas security, checks and display.
