# action_dlattach( )

Downloads an attachment or avatar, and increments the download count. It requires the view_attachments permission. (not for avatars!) It disables the session parser, and clears any previous output. It is accessed via the query string ?action=dlattach. Views to attachments and avatars do not increase hits and are not logged in the "Who's Online" log.
