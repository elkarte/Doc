# db_optimize_table( string $table )

This function optimizes a table.

This function optimizes a table.

  • reclaims storage occupied by dead tuples. In normal PostgreSQL operation, tuples that are deleted or obsoleted by an update are not physically removed from their table; they remain present until a VACUUM is done. Therefore it's necessary to do VACUUM periodically, especially on frequently-updated tables.

Parameters

$table
  • the table to be optimized

Returns

integer
