Package Maillist

Classes summary

Email_Format Attempts, though an outrageous set of assumptions, to reflow/format an email message that was previously wrapped by an email client (so it would look good on a 80x24 screen)
Email_Parse Class to parse and email in to its header and body parts for use in posting
Email_Settings Similar in construction to saveDBSettings,
Emailpost_Controller Post-by-email controller. Handles items pertaining to posting or PM an item that was received by email
ManageMaillist_Controller This class is the administration maillist controller.
