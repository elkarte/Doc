Package Packages

Classes summary

Packages_Controller This class is the administration package manager controller. Its main job is to install/uninstall, allow to browse, packages. In fact, just about everything related to addon packages, including FTP connections when necessary.
PackageServers_Controller PackageServers controller handles browsing, adding and removing package servers, and download of a package from them.
