ManagePosts_Controller ManagePosts controller handles all the administration settings for topics and posts.

Functions summary

approvePosts Approve (or not) some posts... without permission checks...
createPost Create a post, either as new topic (id_topic = 0) or in an existing one.
fixTag Fix a specific class of tag - ie. url with =.
fixTags Fix any URLs posted - ie. remove 'javascript:'.
fixTags_img_callback Ensure image tags do not load anything by themselfs (security)
getFormMsgSubject Prepares a post subject for the post form
lastPost Get the latest post made on the system
modifyPost Modifying a post...
preparsecode Takes a message and parses it, returning the prepared message as a referance.
preparsecode_font_callback Use only the primary (first) font face when multiple are supplied
preparsecode_lowertags_callback Takes a tag and lowercases it
preparsecode_nobbc_callback Ensure tags inside of nobbc do not get parsed by converting the markers to html entities
resizeBBCImages Updates BBC img tags in a message so that the width / height respect the forum settings.
topicSubject Update topic subject.
un_preparsecode This is very simple, and just removes things done by preparsecode.
updateLastMessages Takes an array of board IDs and updates their last messages.
