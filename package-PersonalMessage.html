Classes summary

PersonalMessage_Controller

Personal Message Controller, It allows viewing, sending, deleting, and marking personal messages

Functions summary

addPMRule Add a new PM rule to the database.

deleteMessages Delete the specified personal messages.

loadPMMessageRequest Similar to loadSubjectRequest, this is used by preparePMContext_callback.

loadPMs Load personal messages.

When we are in conversation view, we need to find the base head pm of the conversation. This will set the root head id to each of the node heads
loadPMSearchMessages Gets all the matching message ids, senders and head pm nodes, using standard search only (No caching and the like!)
loadPMSearchResults Loads the actual details of the PM's that were found during the search stage
loadPMSubjectRequest This is used by preparePMContext_callback.
loadRules Load up all the rules for the current user.
markMessages Mark the specified personal messages read.
markMessagesUnread Mark the specified personal messages as unread.
markPMsRead Mark personal messages as read (no new messages) for a particular member.

pmCount How many PMs have you sent lately?
sendpm

Sends a personal message from the specified person to the specified people ($from defaults to the user)
setPMRepliedStatus Used to set a replied status for a given PM.
toggleNewPM Update PM recipient when they receive or read a new PM
updateLabelsToPM Detects personal messages which need a new label.
updatePMLabels Updates PMs with their new label.
updatePMMenuCounts Updates the number of unread messages for a user
updatePMRule Updates a personal messaging rule for the given member.
updatePMRuleAction Updates a personal messaging rule action for the given member.
